\section{Test Execution}

\subsection{Procedure Summary}
\begin{enumerate}
    \item Confirm the setup against the approved bench configuration.
    \item Record DUT revision, instrument IDs, and environmental conditions.
    \item Execute each test case in the agreed sequence.
    \item Save raw captures, processed plots, and anomalies before moving to the next case.
    \item Stop the campaign when a safety or validity condition is violated.
\end{enumerate}

\warningbox{
If an operator deviates from the published procedure, record the deviation
immediately. Silent deviations invalidate otherwise clean-looking result tables.
}

\subsection{Evidence Handling}
\begin{itemize}
    \item Store raw waveform files and screenshots with consistent names.
    \item Link each figure in the report to a trace ID, file path, or bench note.
    \item Separate engineering commentary from direct observation.
    \item Flag any measurement that required manual correction or probe scaling.
\end{itemize}
